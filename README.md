# temperature-humidity-monitoring-system# Temperature & Humidity Monitoring System

An IoT-based environmental monitoring system developed using ESP8266 and DHT11 sensors for real-time temperature and humidity monitoring.

## Features

- Real-time temperature monitoring
- Humidity detection
- Threshold-based automation logic
- Wireless IoT communication
- Sensor-based environmental monitoring

## Technologies Used

- ESP8266
- DHT11 Sensor
- Arduino IDE
- Embedded C++
- IoT Communication
